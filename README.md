# Calculadora IMC

Aplicação console para conta de banco que permite o saque, o depósito e a transferência.

## Funcionalidades

- Cria conta bancária
- Saque
- Depósito
- Transferência

## Tecnologias utilizadas

- dotnet 6.0

## Como executar

Realize o clone do repositório e execute o comando:
```
dotnet run
````
